# NotLetters Desktop

Desktop-приложение на Python для загрузки и просмотра писем из нескольких аккаунтов через API NotLetters.

## Возможности

- хранение нескольких аккаунтов в `accounts.json`;
- добавление, редактирование и удаление аккаунтов;
- импорт аккаунтов из `.txt` в формате `email:password`;
- поиск писем и фильтр по избранному;
- ручное и автоматическое обновление;
- просмотр темы, отправителя, даты, text и html версии письма;
- кеширование писем в памяти по аккаунтам;
- обработка сетевых и API-ошибок без падения UI.

## Структура проекта

- `main.py` - точка входа;
- `models.py` - dataclass-модели;
- `api_client.py` - клиент API NotLetters;
- `storage.py` - JSON-хранилище и импорт из TXT;
- `app_settings.json` - локальные настройки приложения и глобальный API-ключ;
- `ui/app.py` - основной интерфейс;
- `ui/dialogs.py` - формы диалогов.

## Запуск

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

## API-ключ

Приложение поддерживает два варианта авторизации:

- глобальный API-ключ в `app_settings.json` и в кнопке `Настройки API` внутри приложения;
- индивидуальный `Bearer token` у конкретного аккаунта.

Если у аккаунта токен не указан, приложение автоматически использует глобальный API-ключ.
Там же можно настроить пакетное обновление всех аккаунтов: размер пачки и паузу между пачками.

## Хранилище аккаунтов

При первом запуске файл `accounts.json` будет создан автоматически после сохранения первого аккаунта.
Для примера можно скопировать `accounts.example.json` в `accounts.json` и подставить свои данные.
